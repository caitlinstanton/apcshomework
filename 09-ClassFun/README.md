<h2>CLASSES</h2>

- Basechar b1, b2;
- Warrior w1, w2;
- Mage m1, m2;
- b1 = new Basechar();
- w1 = new Warrior();
- m1 = new Mage();
	
- Can't convert instance of subclass to instance of superclass
  - m2 = b1;
- Can't covert instance of one subclass to instance of another subclass
	- m2 = w1;

- Instances of a superclass may become instances of a subclass
